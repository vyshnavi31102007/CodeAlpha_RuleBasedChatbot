🤖 Rule-Based Chatbot

📌 Project Overview

This project is a simple Rule-Based AI Chatbot developed using Python.
The chatbot responds to predefined user inputs using if-elif conditions and provides basic conversational interactions.

It can answer greetings, provide information about AI, ML, Python, and handle simple user commands.

---

✨ Features

- Greeting responses (hello, hi, hey)
- Answer basic questions
- Provides AI, ML, and Python information
- Help command for available options
- Exit command to close chatbot
- User-friendly console interface

---

🛠️ Technologies Used

- Python
- Rule-Based Logic
- Console Application

💬 Available Commands

User Input| Bot Response
hello / hi / hey| Greeting message
how are you| Shows chatbot status
your name| Displays chatbot name
course| Course information
python| Python information
what is ai| AI explanation
what is ml| ML explanation
thank you| Welcome response
help| Shows commands
bye| Exit chatbot
